# Credit Card Fraud Detection System

## Overview
The **Credit Card Fraud Detection System** is a Machine Learning-based project designed to identify fraudulent transactions from large volumes of financial data. The system analyzes transaction patterns and detects anomalies to help prevent financial fraud.

---

## Objective
- Detect fraudulent credit card transactions with high accuracy  
- Minimize false negatives (missed fraud cases)  
- Build a scalable and efficient fraud detection pipeline  

---

## Features
- Data preprocessing and cleaning  
- Handling imbalanced dataset using resampling techniques  
- Exploratory Data Analysis (EDA)  

### Machine Learning Models
- Logistic Regression  
- Random Forest  

### Model Evaluation Metrics
- Accuracy  
- Precision  
- Recall  
- F1-score  

- Fraud prediction system
